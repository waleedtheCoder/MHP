# Mental Health Journal

A personalized mental health journaling platform with AI-powered insights.

## Features
- Daily journaling with sentiment analysis
- Pattern detection and insights
- Smart search and recommendations
- Mood tracking and visualization

## Tech Stack
- **Frontend**: React, Tailwind CSS, Recharts
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **AI**: OpenAI API / Hugging Face

## Setup

### Prerequisites
- Node.js (v16+)
- PostgreSQL (v12+)
- npm or yarn

### Installation

1. Install dependencies:
   ```
   cd server
   npm install
   cd ../client
   npm install
   ```

2. Set up environment variables (see .env.example files)

3. Create database in PostgreSQL

4. Run the project:
   ```
   # Terminal 1 - Backend
   cd server
   npm run dev
   
   # Terminal 2 - Frontend
   cd client
   npm start
   ```
